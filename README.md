## Descripción
Este Trabajo Integrador Final consiste en una serie de algoritmos prácticos enfocados en la resolución de problemas lógicos, estructuras de datos bidimensionales y simulación de redes. 

Los ejercicios incluidos en este proyecto son:
- **AdivinaElNumero:** Juego interactivo para descubrir un número aleatorio entre 1 y 25 con pistas de cercanía.
- **CalcularAreaTriangulo:** Algoritmo matemático simple para obtener el área de un triángulo.
- **CalcularPromedioFinito:** Filtro secuencial que calcula el promedio de hasta 10 números positivos, cortando la ejecución si se ingresa un negativo.
- **ContadorDeVocales:** Analizador de texto que cuenta la cantidad de vocales dentro de una palabra o frase.
- **InteresSimple:** Calculadora financiera que procesa el interés simple en base a un capital, tiempo y tasa precargada.
- **InvertirCadena:** Algoritmo que recibe un texto y lo devuelve completamente invertido.
- **PiramideNumeros:** Generador visual que dibuja una pirámide simétrica y centrada de números naturales correlativos.
- **SumaDeMatrices:** Herramienta bidimensional para cargar dos matrices de 2x2 y calcular su matriz resultante.
- **TablaMultiplicar:** Generador automatizado de la tabla de multiplicar de un número 'n' hasta el 20.
- **RedYComunicacion (Python):** Modelado gráfico y matemático de una red direccionada de 8 nodos con un grado de salida simétrico igual a 2.
